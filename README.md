# Bingo 3X3
### About
Bingo 3X3 is a simple game of chance, which is inspired by [Bingo](http://en.wikipedia.org/wiki/Bingo_(U.S.)). It is written entirely in Python.

### Execution
```Bash
python game.py [bingo_card] [numbers_drawn]
```
